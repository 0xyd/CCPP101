# 0008: *<<* and *>>*  

*n << x* means shift n to left by x bits and *n >> x* vice versa. 

For example, 
```
1 = 0001
1 << 0 = 0001
1 >> 0 = 0000
1 << 1 = 0010 
1 >> 1 = 0000
```
